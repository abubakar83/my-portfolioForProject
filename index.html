01 · About

A developer who cares about the details

I'm Abu Bakar, a passionate Full-Stack Web Developer with a strong focus on building responsive, user-friendly, and high-performance web applications. I enjoy the full journey of a product — translating a design into clean, semantic markup, wiring up smooth interactions, and building the server-side logic and databases that keep everything running reliably.

My toolkit centers on modern JavaScript and the MERN stack — React on the front end, Node.js and Express on the back end, and MongoDB for data — but what drives the work is simpler than any framework: writing code that's easy to maintain, fast to load, and genuinely pleasant for people to use.

Whether I'm building an interface from scratch or improving an existing one, I aim for the same thing every time — software that feels effortless because the hard parts were handled properly underneath.
